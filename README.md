# 🐍 Python Instance Methods Examples

A collection of simple Python programs demonstrating **Instance Methods** in Object-Oriented Programming (OOP).

This project explains how objects, constructors, instance variables, and instance methods work in Python.

## 📌 About The Project

The programs demonstrate different ways of creating and using instance methods:

- Method without parameters
- Method returning a value
- Method with parameters
- Method performing calculations

## ✨ Concepts Covered

✅ Class creation  
✅ Object creation  
✅ Constructor (`__init__`)  
✅ Instance variables  
✅ Instance methods  
✅ Passing arguments to methods  
✅ Returning values from methods

## 📝 Programs Overview

### 1️⃣ Instance Method Without Return Value

- Creates a class with variables
- Calls a method that performs an operation
- Displays the result directly

### 2️⃣ Instance Method With Return Value

- Method performs calculation
- Returns the output
- Result is stored and printed

### 3️⃣ Instance Method With Parameters

- Values are passed while calling the method
- Method performs calculation using given inputs

### 4️⃣ Instance Method Example

- Demonstrates another use of instance methods
- Uses object data and method functionality together
